# LitWeather

A web component using lit-html to display current weather and a forecast with the openWeather apis

## Features:

- Display current weather and forecast for any city using the open weather APIs

## How to use

### Installation

```bash
npm i lit-weather
```

```js
import 'lit-weather/lit-weather.js';
```
